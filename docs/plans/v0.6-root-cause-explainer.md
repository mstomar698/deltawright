## Milestone statement

v0.6 "root-cause explainer" makes Deltawright the single **accurate** place that explains
why a Playwright interaction is in the state it's in. It ships as ONE measured, pure
diagnosis engine over the existing `Delta`, plus a small set of additive, **opt-in**
wrappers; the default `@playwright/test` path stays byte-unchanged and Playwright's
actionability verdict is authoritative and **never overridden** (geometry is annotation,
surfaced as `[geom:]`, and can only *flag* disagreement).

**Non-goals (unchanged):** Deltawright does **not** fix, retry, suppress, or preserve
flakes ‚Äî identifying the cause in one accurate place is the entire value. Pre-flight and
settle-as-a-wait are OBSERVE/EXPLAIN, never flake-suppressants.

**Accuracy is the gating variable.** No diagnosis-emitting capability ships before a
canonical closed taxonomy, a labeled corpus with **mandatory near-miss confusers**, and an
accuracy harness measure it ‚Äî and before the two known silent gaps (two-wave late render,
post-settle JS-recenter stale rect) are closed to **flags**. `"unsure"` is a first-class
output; **silence beats a confidently-wrong label.** All numbers are **corpus-relative**;
real-production precision stays blocked on #25/#41 and is stamped on every output.

## Why the cut (from the design panel's adversary)

On the timing-flake class that dominates a 200-test suite, Deltawright is **today silently
wrong**: a two-wave render with a >120 ms inter-wave gap is reported as a no-op with
`hitMaxWait=false`. Shipping seven diagnosis features on an *unmeasured* engine would make
the "single accurate place" a confidently-wrong place on exactly the failures users most
need explained ‚Äî eroding the trust the product is sold on (the owner's named anti-goal).
So the milestone commits **one measured engine + two ground-truth wrappers + a passive
side-car**, and hard-gates the rest.

## Root-cause taxonomy (closed enum ‚Äî DW-04)

Each code is grounded in a **real primitive signal**; adding/renaming one needs an ADR +
corpus relabel + harness re-run.

| Code | Meaning | Grounding signal |
|---|---|---|
| `covered-by-overlay` | hit-point occupied by another element, PW+geometry agree it blocks | `hitSelf`/`coveredBy` + PW error |
| `off-screen` | outside viewport / scrolled away | `offscreen`/`inViewport=false` |
| `not-visible` | display:none / visibility:hidden / opacity:0 | PW "not visible" |
| `disabled` | control disabled | PW verdict |
| `read-only` | input read-only | PW verdict |
| `pointer-events-none` | computed `pointer-events:none` on target | computed style |
| `unstable-animating` | element not stable (mid-animation) | PW "not stable" |
| `geom-disagreement` | geometry vs PW disagree; carries direction, never overrides | `agreed=false` |
| `background-churn` | high `droppedBackground` may mask the change | `droppedBackground` |
| `detached-re-render` | target removed/replaced by a re-render | ref staleified |
| `settle-timeout` | settle hit the maxWaitMs cap | `hitMaxWait` |
| `suspected-miss-empty` | zero nodes + cap hit; no effect OR missed | empty delta + cap |
| `late-wave-suspected` | structural mutation after settle resolved (gap-E flag) | `lateStructural` |
| `stale-rect-suspected` | post-settle JS reposition moved target after read (gap-F flag) | `stable=false` |
| `injection-blocked` | script injection failed (CSP) | addScriptTag failure |
| `cross-boundary-partial` | cross-origin frame / closed shadow root skipped | traversal skip |
| `pixel-region-fallback` | no DOM delta; screenshot-diff region | screenshot fallback |
| `unknown` | insufficient evidence; first-class "unsure" | catch-all (new codes need ADR) |

## Accuracy strategy (non-circular)

Ground truth is **independent of Deltawright output** ‚Äî scoring NEVER compares against a
stored DW output (that stays the checksum's regression-only job). Each corpus case carries
**three independent oracles**: (1) the **real Playwright action outcome** (the reality
anchor ‚Äî validates the *verdict*, DW-02, not the diagnosis code); (2) a hand-authored
**construction manifest** asserting the intended code; (3) `window.__truth` fixture
instrumentation for hidden causes (wave count/timings, scheduled repositions).

- **Corpus** (`bench/flake-corpus/`): reuse existing fixtures (northstar, gwt, toast,
  stream, livepage, roles, shadow, iframe); **hard requirement** of ‚â•1 fixture per code AND
  ‚â•1 **near-miss confuser** per code (a confuser-free one-fixture-per-code corpus inflates
  precision and is disallowed). ~30‚Äì40 cases; real-app expansion pairs with #25.
- **Harness** (`npm run bench:accuracy`): confusion matrix + per-class P/R/F1 + calibration
  table + ‚Äî as the **headline** ‚Äî a **silent-miss rate** (ground-truth causes DW neither
  diagnosed nor flagged).
- **CI floors:** verdict-vs-reality agreement = **100%** (DW-02); confirmed-band macro
  precision **‚â• 0.95** (the only band allowed a high claim); silent-miss **‚â§ 0.05**
  (achievable only after the two gaps are closed to flags).
- **Confidence** (one shared type): `confirmed` (an authoritative engine named the cause ‚Äî
  PW error string, or geometry+PW agreement) | `suspected` (geometry-only or a timing
  heuristic) | `unknown`/unsure. Emit suspected/unknown over a confident wrong code.
- **Honesty stamp** on every output: numbers are corpus-relative ‚Äî "DW diagnoses correctly
  on a DOM we assert resembles the failure mode," NOT real production precision.

## Guards & watches

- **DW-02** (verdict matches reality) stays active ‚Äî protected by the reality-oracle 100%
  CI gate + agree-or-flag unit tests (a future `diagnose()` edit must never contradict PW).
- **DW-03** (new) ‚Äî a diagnosis is a *hypothesis* carrying confidence; must emit "unsure"
  over a guess and must never contradict Playwright's verdict.
- **DW-04** (new) ‚Äî the taxonomy is closed + versioned; code changes need ADR + corpus
  relabel + harness re-run.
- A **default-path checksum test** proves the new diagnostics-render flag and new
  `DeltaStats`/`GeometryRead` fields (`lateStructural`, `stable`) don't change default bytes.
