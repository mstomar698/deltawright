## A. Tests / build / packaging / CI

- **‚úÖ FIXED [HIGH] Flaky timing-assertion family (5 sites).** The `framework-quiescence` flake (241ms
  under load) was one of five identical `settleMs < BUSY_MS-30` (~220ms ceiling, ~90ms headroom)
  assertions ‚Äî also `framework-quiescence.spec.ts:30`, `quiescence.spec.ts:28`, `quiescence.spec.ts:79`,
  `wait.spec.ts:109`. The default settle (~130-160ms, `DEFAULT_SETTLE.quietMs=120`) is independent of
  the busy window, so **widening the busy/delay window 250‚Üí600ms** (fixtures + spec constants) enlarges
  the plain-vs-busy margin to ~320ms ‚Äî robust under load. (Alternative considered: assert the *relative*
  invariant `plain.settleMs < q.settleMs`.)
- **‚úÖ FIXED [MED] `release.yml` has no `concurrency:` group** ‚Äî the `npm view ‚Ä¶ && skip` guard is
  TOCTOU-racy across concurrent runs (two tag pushes both pass "not published" ‚Üí both publish ‚Üí the
  loser fails red). Added `concurrency: { group: release-${{ github.ref }}, cancel-in-progress: false }`.
- **[LOW] `ci.yml` has no `concurrency:` group** ‚Üí rapid PR pushes run redundant CI (cost only). Add
  `group: ci-${{ github.ref }}, cancel-in-progress: true`.
- **[LOW] Supply-chain hygiene:** actions pinned to floating major tags (`@v7`) not commit SHAs;
  `release.yml` runs unpinned `npm install -g npm@latest`. Worth SHA-pinning for an OIDC publisher.
- **[LOW] `bg-churn.spec.ts:24`** (`recurringInsert >= 4` in a 400ms cap) has margin but is theoretically
  load-sensitive. (The `effect-settle`/`northstar` `appearedMs >=` lower bounds get *safer* under load.)

### ESM/CJS dual build ‚Äî VERDICT: **NEEDS-CARE** (feasible, not a drop-in) ‚Äî DEFERRED (design decision)
`tsup format:['esm','cjs']` works **only with `shims:true`** and only if the bins stay ESM:
1. Without shims, `import.meta.url` is `undefined` in CJS ‚Üí the top-level `const new URL('./injected/
   observer.global.js', import.meta.url)` in `inject.ts` throws at import time (`require('deltawright')`
   fails to load). Same for `cli.ts`. ‚Üí `shims:true` mandatory.
2. tsup's shim uses `new URL('file:'+__filename)` (not `pathToFileURL`) ‚Üí a path with spaces can
   mis-resolve the observer candidate. Patch the shim to `pathToFileURL`.
3. The "is main module" check (`mcp/server.ts`) compares encoders that differ under shims ‚Üí dual-build
   only the 6 **library** entries; keep the `deltawright`/`deltawright-mcp` **bins ESM**.
4. Cross-format dynamic `import()` + the dual-package hazard are **low risk** (DW's singleton lives in
   `window.__deltawright`, not Node module state).
There is a working consumer workaround today: `await import('deltawright')` from CJS.

## B. Core delta / observe engine

- **‚úÖ FIXED [MED] `live-routing.ts` ‚Äî over-broad client-abort exclusion dropped a real backend fault.**
  `/ERR_ABORTED|aborted/i` also matched `net::ERR_CONNECTION_ABORTED` (a connection dropped by the
  network/server), silently excluding a genuine infra fault. Now `/\bERR_ABORTED\b/i` (only the client
  sentinel). Regression test added.
- **‚úÖ FIXED [MED] `actAndObserve.ts` ‚Äî child-frame late-watch MutationObserver leaked.** With
  `frames:true` + `lateWatchMs>0`, each child frame started a `lateObserver` never torn down (only the
  main page's `lateResult()` runs), and the frame's late signal is discarded anyway. Now child frames
  get `{ ...settle, lateWatchMs: 0 }`.
- **[MED] `observer.ts:~108` ‚Äî synchronous XHR permanently leaks the in-flight counter.** For a sync
  `XMLHttpRequest`, `loadend` fires *during* `send()` (before `apply` returns), so the `{once:true}`
  listener is attached too late and never decrements ‚Üí `isQuiescent()` stuck `false` for the page's life
  ‚Üí every `awaitQuiescence` settle waits to the cap. Sync XHR is rare/deprecated, and the fix is
  intricate (increment+listen before apply, but then undo on a sync throw), so DEFERRED. **Fix:**
  attach+increment before `apply`, decrement in a `catch` on a synchronous throw.
- **[LOW] `actAndObserve.ts` ‚Äî `awaitQuiescence` never armed inside child frames** (`enableQuiescence()`
  runs on the main page only). A child frame's `isQuiescent()` is always true ‚Üí the frame silently skips
  the network gate. Fix per-frame or document as main-frame-only.
- **[LOW] `observer.ts` `settleAnimations` ‚Äî infinite/paused animations burn the full `animMaxMs`.** A
  looping spinner's `.finished` never resolves ‚Üí the timeout always fires. **Fix:** exclude
  infinite-iteration + `paused` animations from the awaited set. (Behavior change ‚Äî validate against the
  animation tests.)
- **[LOW] `screenshot-diff.ts` ‚Äî pixel-region rect is in device pixels on HiDPI** (`deviceScaleFactor>1`),
  inconsistent with CSS-pixel DOM geometry. Confined to the opt-in `screenshotFallback` empty-DOM path.

### Clean: actionability reconcile (DW-02/03), input-integrity classify, coalesce, checksum, live-routing
teardown/privacy, anchor-rescue, baseline lifecycle, shadow-root re-observation.

## C. New primitives (pageMap / effect-settle / scoreSelectors)

- **‚úÖ FIXED [MED] `observeEffectSettled` leaked the baseline footprint.** The delta path clears
  `bg{Text,Attr,Insert,Remove}` in `collect()`, but `observeEffectSettled` cleans up with `reset()`,
  which never touched them ‚Üí a later `{baseline:false}` call (contra its doc) ran against stale
  background. Now `waitForEffectSettle` consumes the footprint in a `finally` (symmetric with `collect`;
  NOT in `reset()`, which `arm()` calls after `sampleBaseline`). Regression test added.
- **‚úÖ FIXED [LOW] `pageMap` reconcile flagged spurious `geomDisagreesWithPlaywright` on off-screen
  nodes** ‚Äî a trial-click auto-scrolls an off-screen node into view and passes (fabricating a
  disagreement + mutating the scroll position `scan` already captured). Off-screen nodes now take the
  geometry path under reconcile (authoritatively NOT-actionable-here; no probe, no scroll). Regression
  test added.
- **‚úÖ FIXED [MED] A removal-dominated effect seeds the region from the removed node's PARENT.** Closing a
  modal appended to `<body>` ‚Üí `m.target === body` ‚Üí region ‚âà whole page ‚Üí PHASE 2 every mutation
  intersects ‚Üí `hitMaxWait` on a live page (the global-quiescence failure R1 avoids). Resolved by the
  "report region=null + effectAppeared=true for top-level removals" option: `<body>`/`<html>` effect
  targets are now **unlocalizable** ‚Äî they latch `effectAppeared` but never seed the region or reset the
  quiet timer (`isUnlocalizable` in `observer.ts`), so a top-level removal settles cleanly with
  `region: null` while a localizable follow-on still scopes the region. Tests (e)/(f) in
  `effect-settle.spec.ts`.
- **[LOW/MED] Multiple/nested overlays collapse to one `layer 1` and can carry the WRONG label** ‚Äî a
  `role=alert` toast + a modal ‚Üí a node in overlay B labeled with overlay A's name. Track the label per
  overlay root.
- **[LOW] Sticky/fixed page chrome is classified as an "overlay" layer** ‚Äî a sticky header covering
  scrolled content reports as `layer 1 overlay`. Add a heuristic (full-width top-anchored sticky ‚âÝ modal).
- **[LOW] `appearedMs` is stamped at drain time, not mutation time** (MutationRecords carry no
  timestamp) ‚Äî soften the doc wording.

### Clean: `scoreSelectors` (synthesized geom candidate can never be `bestDurable`; escaping/round-trip
guarding/status ladder/concurrency sound), `isGeneratedId`/`isDynamicText`, `zoneOf`.

## D. Offline / reporting / MCP surfaces

every dynamic string through `escapeHtml` (no XSS breakout from a malicious side-car); reporter
`sanitize()` blocks path traversal; the zip parser validates offsets/bounds + refuses
ZIP64/encryption + tolerates truncation; confidence/taxonomy honesty invariants hold. Only:

- **‚úÖ FIXED [LOW] `mcp/server.ts` ‚Äî `serverInfo.version` hardcoded `'0.1.0'`** while the package is
  0.9.3. Now reads the real version from `package.json` (like `cli.ts`).
- **[LOW] `read-trace.ts:~85` ‚Äî `harnessBucket` can mislabel a 4xx line as `5xx/gateway`** when the line
  carries an incidental 5xx-shaped number (`404 (took 503 ms)`). Routing decision unaffected (cosmetic
  label). Classify off the matched status token, not a free-floating `\b5\d\d\b`.
- **[LOW] `read-trace.ts:~228` ‚Äî `capText` can split a surrogate pair** (slice by UTF-16 unit) ‚Üí a lone
  `ÔøΩ`. Slice by code points.
